### Use LLMs locally on your hardware

All in one LLM+desktop:

* LMstudio: https://lmstudio.ai/
* GPT4all: https://www.nomic.ai/gpt4all

Run your local LLM API endpoint to configure in tools
  * Ollama: https://ollama.com/
  * Llama.CPP (also runs non-llama models): https://github.com/ggml-org/llama.cpp

Run a local Web frontend that uses your local LLM API endpoint
   * OpenWebUI: https://docs.openwebui.com/
   * Using OpenWebUI on Llama.CPP:  https://github.com/ggml-org/llama.cpp/discussions/16938

Image Generation with your local LLM
   * ComfyUI: https://comfy.org/download
     
Speech To Text with your local LLM
   * Whisper https://github.com/ggml-org/whisper.cpp 

Codeing Agents
   * https://opencode.ai/
